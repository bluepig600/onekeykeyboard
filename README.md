# OneKeyKeyboard
!!Must be run on windows computer!!

This is a keyboard python application that lets you type the full alphabet using only the space bar.
## How to install
1. Install pynput using pip
```
pip install pynput
```
2. Download main.py from repo
3. Run main.py
## How to use
1. Wait for it to intalize.
2. Pressing the space key once types the character displayed in the window.
3. Pressing and holding the space key cycles through the characters.
4. Double pressing the space key advances one character.
5. Triple pressing the space key retreats one character.
